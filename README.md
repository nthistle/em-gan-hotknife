# em-gan-hotknife

GAN to generate artificial electron microscopy data to "fill in" the cut left by hot knife in FIB-SEM

## Conda packages what **should** be necessary
(python 3.6.6 w conda)

pandas
pillow
scipy
z5py
h5py
tensorflow-gpu
keras
